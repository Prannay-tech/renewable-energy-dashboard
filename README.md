## Overview

Built for energy analysts and investors who need to evaluate solar project economics against real market conditions. All financial outputs are grounded in live government API data — not hardcoded assumptions.

---

## Features

### 📊 Tab 1 — Market Overview
- Live electricity prices and renewable capacity from **EIA Open Data API**
- Federal Funds Rate and CPI inflation from **FRED API**
- AreaChart + BarChart visualisations with skeleton loading states
- Market context cards with dynamic text (e.g. rate environment analysis)

### ⚡ Tab 2 — Project Economics Calculator
- **IRR, NPV, LCOE, payback period** — full 20-year discounted cash flow model
- Three preset scenarios: Base Case, Optimistic, Conservative
- Sensitivity heatmap — IRR across electricity rate × capacity factor grid
- **AI Investment Memo** — one-click structured memo generated by Claude Sonnet

### 🗺️ Tab 3 — Geographic Visualization
- Interactive Leaflet map — all 50 states with EIA electricity price overlay
- Solar irradiance overlay from **NREL** data
- Click any state → push its electricity rate directly into the calculator
- State rankings sidebar

### 🤖 Tab 4 — Research Assistant
- Streaming chat powered by **Claude claude-haiku-4-5**
- Every response grounded in live EIA + FRED data from your session
- "What AI sees" context panel — full transparency on what's in the model's context
- Source labels on every response: 🟢 EIA Live · 🟡 FRED Live · 🔢 Calculated

## Architecture

All API keys are server-side only — accessed through Next.js `/api/*` route proxies. No keys exposed to the browser.

```
Browser (React/Zustand)
    ↕ fetch
Next.js API Routes (server-side)
    ↕ authenticated requests
EIA API · FRED API · NREL API · Anthropic API
```

**Cross-tab data flows (via Zustand):**
1. **Geographic → Economics** — click a state, its electricity rate auto-fills the calculator
2. **Economics → AI** — every chat message includes your full scenario + computed results
3. **Market → AI** — live EIA/FRED snapshot injected into every AI request

## Local Setup

```bash
git clone https://github.com/Prannay-tech/renewable-energy-dashboard.git
cd renewable-energy-dashboard/src
npm install
```

Create `src/.env.local`:
```env
EIA_API_KEY=your_key       # https://www.eia.gov/opendata/
FRED_API_KEY=your_key      # https://fred.stlouisfed.org/docs/api/
NREL_API_KEY=your_key      # https://developer.nrel.gov/ (DEMO_KEY works for testing)
ANTHROPIC_API_KEY=your_key # https://console.anthropic.com/
```

```bash
npm run dev
# → http://localhost:3000
```

## Financial Model

- **IRR** — bisection method (robust for non-monotonic cash flows)
- **NPV** — 8% discount rate (configurable)
- **LCOE** — discounted cost / discounted energy over 25-year project life
- **ITC** — 30% Investment Tax Credit shown as net cost reduction
- **Debt structure** — 70% leverage at configurable interest rate
- **Degradation** — 0.5% annual output degradation
